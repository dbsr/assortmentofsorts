Help

With this addon you can group your addons into different profiles and (de)activate them with a mouseclick. Because addons deserve profiles too.

The default profile is the only profile you cannot remove or delete. Addons selected in this profile will always be loaded together with the addons for the profile that is currently activated."

About

Why?
I made this addon because I noticed that certain addons I use for web development use a lot of memory/cpu. I don't need those addons all the time but I found no setting or addon that provided a simple way to activate / deactivate a group of addons.
How?
This addon is completely written in coffeescript. For the css it uses less. It makes use of the following external libraries: bootstrap and JQuery.
